🌌 Solar System Explorer (Interactive 3D)
An interactive 3D visualization of the Solar System built using Three.js.
This project simulates the Sun, planets, moons, orbital motion, and planetary information in a realistic space environment.
Users can rotate the camera, zoom, change simulation speed, and click planets to explore detailed scientific data.
🚀 Features
🌍 Interactive 3D Solar System
Realistic Sun and planets rendered using Three.js
Planets orbit the Sun in elliptical orbits
Realistic relative orbital speeds
🪐 Planetary Details Panel
Click any planet to view:
Distance from Sun
Orbital Period
Orbital Velocity
Eccentricity
Diameter
Mass
Surface Gravity
Rotation Period
Temperature
Number of moons
Atmospheric composition
Scientific data sourced from NASA NSSDC Planetary Fact Sheet.
🌠 Space Environment
14,000+ background stars
Asteroid belt simulation
Saturn ring system
Planetary atmospheres and glow effects

🎮 User Controls
Control	Function
